Данный репозиторий описывает консольную утилиту отображающая метрики аппаратной конфигурации в момент запуска и на протяжении определенного временного промежутка.

В настоящий момент получаемые данные:
1. Средняя загрузка CPU (Load average)
2. Кол-во ОЗУ, общее / используемое, доступное 
3. Пропускную способность сетевого интерфейса (для WiFi адаптера на macOS не выдает данные ввиду особенности ОС(для LAN подключение не проверялось, для VPN адаптера работает корректно.))

---

###Для запуска необходимо:

- Запустить файл main.go из директории /cmd/cli/main.go
